## Java Study

### 자바 공부 자료

### 프로젝트

1. __HelloWorld__
   Hello World 출력 프로젝트

2. __Network__
   자바 소켓 프로그래밍.
   서버와 클라이언트 간의 통신 구현

3. __Network-Dict__
   자바 소켓 프로그래밍 응용.
   서버와 클라이언트가 영단어와 뜻을 주고 받는 검색 프로그램

4. __BankStatement__
   입출금 내역 분석 프로그램.

   SRP(단일책임원칙)과 테스트 도구인 JUNIT을 학습
